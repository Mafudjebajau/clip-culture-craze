
Esta pasta contém vídeos de exemplo para o aplicativo TikTok clone.

Para adicionar seus próprios vídeos de exemplo:
1. Adicione arquivos MP4 nesta pasta com nomes como video1.mp4, video2.mp4, etc.
2. Os vídeos devem ser curtos (15-60 segundos) e em formato compatível com navegadores web
3. Estes vídeos serão usados como fallback quando os URLs remotos não funcionarem

Nota: Esta pasta já contém alguns arquivos de vídeo de amostra para demonstração.
